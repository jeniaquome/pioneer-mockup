## Run the application using docker compose
In a new terminal window, navigate to the application directory

Then, with docker compose, start the application
```
docker compose up --build
```

## Open the Fast API application in the browser

You can see the API documentation at
[http://localhost:8000/docs](http://localhost:8000/docs)


## Using the Fast API endpoints

You can use an API test client like Insomnia
https://insomnia.rest/download

Or, you can use the "Try it out" methods in the docs URL above.
